
# 🎉 Diwali Sales Analysis System

🚀 A Python-based data analytics project that analyzes Diwali season sales data to uncover customer behavior, product demand, and business insights.

---

## 📌 Project Overview
Festival seasons bring significant changes in customer purchasing behavior.  
This project focuses on analyzing sales data to identify trends, understand customer segments, and generate insights that can help businesses improve decision-making.

---

## 🎯 Objectives
- Analyze customer buying patterns during festival seasons  
- Identify top-selling products and categories  
- Understand the impact of discounts on sales  
- Discover high-value customer segments  
- Generate data-driven business insights  

---

## 🛠️ Tech Stack
- **Language:** Python 🐍  
- **Libraries:** Pandas, NumPy, Matplotlib, Seaborn  
- **Tools:** Jupyter Notebook  

---

## 📊 Project Workflow
1️⃣ Data Collection  
2️⃣ Data Cleaning & Preprocessing  
3️⃣ Exploratory Data Analysis (EDA)  
4️⃣ Data Visualization  
5️⃣ Insight Generation  

---

## 📁 Dataset Description
The dataset includes:
- Customer demographics (Age, Gender, State, Occupation)  
- Product details (Category, Product ID)  
- Transaction details (Purchase Amount, Orders)  
- Festival/seasonal purchase data  

---

- 🎯 Identified top-performing product categories during festivals
